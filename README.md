# The F2E week3 - 全台公車動態時刻查詢應用服務

## 使用技術

- React
- Chakra-UI
- React redux
- redux-thunk
- leaflet

## 系統說明

- 安裝： `npm install`
- 執行： `npm start`

Node 版本: v16

## 資料夾說明

api - 存放所有呼叫 api 的函式

components - 共用元件

constant - 常數資料

hooks/utils - 自訂函式

store - redux 資料夾

- [x] Home - 首頁
- [ ] Nearby - 附近站牌
- [ ] Search - 搜尋頁面
- [ ] Detail - 公車資訊頁面
- [ ] Recommend - 推薦(尚未切版)
